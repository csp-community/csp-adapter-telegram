## Pre-requisites

You need Python >=3.10 on your machine to install `csp-adapter-telegram`.

## Install with `pip`

```bash
pip install csp-adapter-telegram
```

## Install with `conda`

```bash
conda install csp-adapter-telegram --channel conda-forge
```

## Source installation

For other platforms and for development installations, [build `csp-adapter-telegram` from source](Build-from-Source).
